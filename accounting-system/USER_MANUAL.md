# คู่มือการติดตั้งและการใช้งานระบบบัญชีองค์กร (Antigravity Ledger Manual)

ยินดีต้อนรับสู่ระบบบัญชีบริหารจัดการนิติบุคคล ระบบนี้ถูกสร้างขึ้นแบบ **Local-first (Serverless)** ข้อมูลทุกอย่างจะถูกบันทึกอย่างปลอดภัยในบราวเซอร์ของคุณเองโดยใช้ฐานข้อมูล IndexedDB ซึ่งช่วยปกป้องความลับทางการเงินสูงสุดและรองรับการจัดเก็บข้อมูลธุรกรรมขนาดใหญ่กว่า 100,000 รายการต่อปี

---

## ภาคที่ 1: คู่มือการติดตั้งและเปิดใช้งาน (Installation Guide)

เนื่องจากระบบนี้ใช้เทคโนโลยี ES Modules ในเบราว์เซอร์ การดับเบิ้ลคลิกไฟล์ HTML ตรงๆ อาจทำให้เบราว์เซอร์บางตัวบล็อกการทำงาน (CORS Policy) ดังนั้นจำเป็นต้องรันผ่านเว็บบอร์ดจำลอง (Local HTTP Server) ซึ่งทำได้ง่ายๆ ดังนี้:

### สำหรับผู้ใช้งานทั่วไป (ง่ายที่สุดด้วย Python - ติดตั้งมาพร้อม macOS)
1. เปิดโปรแกรม **Terminal** บน Mac ของคุณ
2. พิมพ์คำสั่งเพื่อเข้าไปยังโฟลเดอร์ของโปรแกรม:
   ```bash
   cd /Users/user/Documents/accounting-system
   ```
3. รันคำสั่งเปิดเซิร์ฟเวอร์จำลอง:
   ```bash
   python3 -m http.server 8080
   ```
4. เปิดเว็บเบราว์เซอร์ (แนะนำ Google Chrome หรือ Microsoft Edge) แล้วพิมพ์ URL:
   👉 **[http://localhost:8080](http://localhost:8080)**

## ภาคที่ 2: คู่มือการใช้งานแต่ละเมนู (User Manual)

### 1. แผงควบคุม (Dashboard)
* **การแสดงผล:** หน้าแรกจะรวมข้อมูลสำคัญ เช่น ยอดเงินสดคงเหลือในธนาคาร, ยอดรายได้สะสม, ค่าใช้จ่ายสะสม และกำไรสุทธิคาดการณ์
* **แผนภูมิวิเคราะห์:** มีกราฟแสดงเปรียบเทียบ รายรับ-รายจ่าย รายเดือน และกราฟวงกลมแยกหมวดหมู่ค่าใช้จ่าย
* **ประวัติล่าสุด:** ตารางสรุป 5 รายการบัญชีล่าสุดที่เกิดขึ้น

### 2. ผังบัญชี (Chart of Accounts)
* **การใช้งาน:** แสดงรายชื่อรหัสบัญชี 5 หมวดหลัก (สินทรัพย์, หนี้สิน, ส่วนของทุน, รายได้, ค่าใช้จ่าย) 
* **การเพิ่มบัญชี:** คลิกปุ่ม **"เพิ่มบัญชีใหม่"** ระบุรหัสบัญชี 6 หลักตามเกณฑ์สรรพากร ชื่อภาษาไทย และเลือกหมวดบัญชี
* **ความปลอดภัย:** บัญชีหลักของระบบจะถูกบล็อกไม่ให้ลบ เพื่อป้องกันการทำงานของตรรกะระบบบัญชีอัตโนมัติเสียหาย

### 3. สมุดรายวันทั่วไป (Journal Entries)
* **การใช้งาน:** ใช้สำหรับดูประวัติการลงบัญชีทั้งหมดในองค์กร คัดกรองตามวันที่ หรือค้นหาคำอธิบาย
* **การลงบัญชีเอง (Manual Journal Entry):**
  1. คลิกปุ่ม **"บันทึกรายวันทั่วไป"**
  2. ระบุ วันที่, เลขที่เอกสารอ้างอิง, และคำอธิบายรายการ
  3. เลือกบัญชีเดบิตและเครดิต ระบุตัวเลขเงิน
  4. คลิก **"เพิ่มแถว"** เพื่อระบุคู่บัญชีที่เหลือ
  5. **สำคัญ:** ยอดเดบิต (Dr.) รวม และเครดิต (Cr.) รวมต้องเท่ากัน (ต่างกัน 0.00) ปุ่มบันทึกจึงจะยอมให้กดใช้งานได้

### 4. บัญชีแยกประเภท (General Ledger)
* **การใช้งาน:** เลือกบัญชีที่ต้องการตรวจสอบจากรายการดรอปดาวน์ และเลือกช่วงวันที่ที่ต้องการดู
* **ผลลัพธ์:** ระบบจะคำนวณและแสดงยอดคงเหลือยกไปเรียงตามแต่ละบรรทัดให้เห็นความเคลื่อนไหวทางเข้าออกของเงินในบัญชีนั้นๆ

### 5. งบทดลอง (Trial Balance)
* **การใช้งาน:** หน้าจอสรุปยอดเดบิตและเครดิตสะสมของทุกบัญชี เพื่อทดสอบสมดุลบัญชี (ยอดรวมทั้งสองฝั่งต้องเท่ากันเสมอ)

### 6. งบกำไรขาดทุน (Profit & Loss Statement)
* **การใช้งาน:** ดึงสถิติรายได้หักค่าใช้จ่าย เพื่อคำนวณกำไรสุทธิจากการดำเนินงานในช่วงเวลาที่เลือก

### 7. งบแสดงฐานะการเงิน (Balance Sheet)
* **การใช้งาน:** แสดงสมการหลักทางการเงิน **สินทรัพย์ = หนี้สิน + ส่วนของผู้ถือหุ้น**
* *หมายเหตุ:* ในส่วนของทุน ระบบจะคำนวณ "กำไรสะสมรวมกำไรสุทธิงวดปัจจุบัน" ให้อัตโนมัติ เพื่อให้งบดุลลงตัว

### 8. รายงานภาษีซื้อ ภาษีขาย และภาษีหัก ณ ที่จ่าย (Tax Reports) - [เพิ่มเติม]
แท็บรายงานแบบทางการสำหรับเตรียมแบบยื่นส่งต่อสรรพากร:
* **รายงานภาษีขาย (Sales VAT Report):** แสดงประวัติภาษีที่เก็บจากลูกค้า โดยแยกมูลค่าสินค้า/บริการ และภาษี 7% ยอดรวมสุทธิท้ายตาราง
* **รายงานภาษีซื้อ (Purchase VAT Report):** แสดงรายการภาษีซื้อที่ได้บันทึกไว้จากการตั้งหนี้/จ่ายค่าสาธารณูปโภค 
* **รายงานภาษีหัก ณ ที่จ่าย (Withholding Tax Report):** สรุปยอดภาษีหัก ณ ที่จ่าย 1%, 3%, 5% แสดงแถบสถานะชัดเจนว่า "เราหักผู้อื่นไว้" (เพื่อนำยื่นแบบ ภ.ง.ด.3/53) หรือ "เราถูกหักไว้" (เพื่อใช้เคลมขอคืนภาษีตอนสิ้นปี)
* **การพิมพ์รายงาน:** เลือกช่วงวันที่ที่ต้องการออกรายงาน เลือกแท็บรายงานที่ต้องการ แล้วกดปุ่ม **"พิมพ์รายงานนี้ (Print A4)"** ด้านบน บราวเซอร์จะซ่อน Sidebar/Header และขยายตารางภาษีแนวขวางเต็มกระดาษ A4 Landscape โดยอัตโนมัติ

### 9. ออกเอกสารใบเสร็จ/ใบแจ้งหนี้ และ บันทึกค่าใช้จ่าย (Invoices & Bills)
นี่คือฟีเจอร์อัจฉริยะที่ช่วยประหยัดเวลาการลงสมุดรายวันด้วยตนเอง:
* **การออกใบแจ้งหนี้ (Invoices):** 
  - กรอกชื่อลูกค้า, วันที่, รายละเอียด และราคา
  - **ระบบภาษี:** สามารถติ๊กเลือกอัตรา VAT (7%) และภาษีหัก ณ ที่จ่าย (3% สำหรับค่าบริการ)
  - เลือกสถานะ: **"ค้างชำระ"** (ระบบจะลงบัญชีเป็นลูกหนี้การค้า) หรือ **"ชำระแล้ว"** (ระบบจะลงบัญชีเป็นเงินสด/ธนาคารทันที)
  - ระบบจะบันทึกเอกสารและโพสต์รายการบัญชีเดบิต/เครดิตภาษีเข้าสมุดรายวันทั่วไปให้โดยอัตโนมัติ
  - กดปุ่ม **"พิมพ์"** เพื่อแสดงตัวอย่างใบกำกับภาษีตามแบบมาตรฐาน พร้อมกดสั่งพิมพ์หรือบันทึกเป็น PDF ได้เลย
* **การบันทึกบิลค่าใช้จ่าย (Bills):**
  - กรอกชื่อผู้ให้บริการ เลือกหมวดหมู่รายจ่าย (เช่น ค่าไฟ, ค่าเช่า, เงินเดือน) และระบุราคา
  - สามารถเลือกหัก ณ ที่จ่ายที่เราหักผู้ให้บริการเก็บไว้ (เช่น ค่าเช่า 5%, บริการ 3%) เพื่อนำส่งสรรพากร
  - ระบบจะลงบัญชีอัตโนมัติในฝั่งรายจ่ายทันที

### 10. ตั้งค่าระบบและจัดการสำรองข้อมูล (Settings & Backup Tools)
* ** Company Profile:** แก้ไขชื่อนิติบุคคล เลขประจำตัวผู้เสียภาษี 13 หลัก และที่อยู่สำนักงานใหญ่ เพื่อไปปรากฏในใบแจ้งหนี้และรายงาน
* **สำรองข้อมูล (Export JSON):** ดาวน์โหลดข้อมูลทั้งหมดออกมาเป็นไฟล์เก็บไว้ในคอมพิวเตอร์ของคุณเพื่อความปลอดภัย
* **กู้คืนข้อมูล (Import JSON):** นำไฟล์สำรองกลับมาอัปโหลด ยอดเงินและธุรกรรมเก่าจะกลับคืนมาทั้งหมด
* **ล้างข้อมูล (Clear Database):** ล้างข้อมูลทั้งหมดเพื่อขึ้นปีงบประมาณใหม่หรือเริ่มใช้จริง
* **ทดสอบประสิทธิภาพ (Stress Test):** สั่งให้บราวเซอร์จำลองธุรกรรม **100,000 รายการ** เพื่อทดสอบความแรงของระบบ

## ภาคที่ 3: คำแนะนำและการบำรุงรักษา (Maintenance Tips)

1. **การสำรองข้อมูลรายสัปดาห์/รายเดือน:** แนะนำให้กดปุ่ม **Export JSON** เก็บไว้เสมอ เผื่อกรณีคอมพิวเตอร์มีปัญหา หรือมีการล้างประวัติการท่องเว็บ ข้อมูลจะได้ไม่สูญหาย
2. **การย้ายเครื่องทำงาน:** เพียงคัดลอกไฟล์โฟลเดอร์ `accounting-system` ไปเครื่องใหม่ รันคำสั่งไพธอนเหมือนเดิม และกดนำเข้าไฟล์ JSON ล่าสุดที่คุณสำรองไว้
3. **การเข้าใช้งานร่วมกันหลายคน:** เนื่องจากข้อมูลเก็บในเบราว์เซอร์ของแต่ละเครื่อง หากต้องการแบ่งปันข้อมูล ให้ทำการส่งไฟล์ JSON สำรองให้เพื่อนร่วมงานนำเข้ามาอัปเดตในเครื่องของเขาได้เลยครับ
